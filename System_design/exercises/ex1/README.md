Программа реализует создание системы управления спутниками:
Satellite - абстрактный класс спутников;
ImagingSatellite - наследник Satellite, ДЗЗ - осуществляющий снимки Земли;
CommunicationSatellite - наследник Satellite, Связь - осуществляющий
SatelliteConstellation - спутниковая группировка, она связана со спутниками с помощью агрегации.

В файле Main.java осуществляется:
- создание 2 спутников связи и 3 спутников ДЗЗ;
- создание и формирование группировки 'RU Basic';
- активация спутников;
- выполнение миссий группировки.
