Вспоминаем задачу

Тебе нужно создать систему управления спутниками, о которой говорится в видео. Это поможет увидеть принципы ООП в действии.
В качестве решения необходимо подгрузить консольный вывод твоей программы (в виде скриншота), а также папку с проектом или ссылку на гит-репозиторий

📍Обрати внимание 

  • Баллы за семинар не начисляются, но выполнение всех семинаров дает суммарно 15 баллов при зачете
  • На выполнение семинара отводится календарный месяц

Диаграмма классов

Ниже — сформированная диаграмма классов. Она соответствует архитектуре представленной космической системы. При написании кода стоит опираться именно на нее

На практике с помощью диаграмм классов разработчики говорят друг другу, что и как нужно реализовать

Рассмотрим каждый класс отдельно

1) Класс Satellite

Это главный узел нашей системы

Атрибуты

 name: String  Название
 isActive: boolean  Флаг активности
 batteryLevel: double  Уровень заряда

Методы

 activate(): boolean  Метод включения спутника (спутник включается, если уровень заряда больше 0.2)
 deactivate(): void  Метод выключения спутника (только если он был включен)
 consumeBattery(double): void

 Метод потребления энергии спутником при выполнении миссии

  • Принимает на вход количество потребляемой энергии и вычитает из общей
  • Если ее остается критически мало, то выключает спутник
 performMission(): void

Метод выполнения миссии — его нужно сделать абстрактным, чтобы всем наследникам нужно было переопределить миссию

Что учесть: метод имеет доступ видимости protected, чтобы к нему можно было обращаться только через наследников

2) Класс ImagingSatellite

Этот класс является наследником Satellite. Он соответствует спутнику дистанционного зондирования Земли (ДЗЗ)

Атрибуты

 resolution: double  Разрешение получаемых со спутника фото
 photosTaken: int  Счетчик полученных фото

Методы

 getResolution(): double  Геттер для получения скрытого поля resolution
 getPhotosTaken(): int  Геттер для получения скрытого поля photosTaken
 performMission(): void

 Переопределенная функция Satellite. Если спутник включен, нужно сделать фото с помощью takePhoto, а также потратить 0.08 заряда
 takePhoto(): void  Метод, который увеличивает счетчик фотографий, сделанных спутником, при условии, что спутник активен

Не забудь переопределить методы:
В Java это можно сделать с помощью аннотации @Override

  • метод performMission() (с созданием новых методов для каждого спутника в зависимости от его предназначения);

  • метод toString() для красивого вывода (иначе будет выводиться ссылка на память компьютера).

3) Класс CommunicationSatellite

Этот класс также является наследником Satellite. Он соответствует спутнику связи в нашей системе

Атрибуты

bandWidth: double  Полоса пропускания канала связи

Методы

 getBandwidth(): double  Геттер для получения скрытого поля bandWidth
 performMission(): void

 Переопределенная функция Satellite. Если спутник включен, нужно отправить данные с помощью sendData, а также потратить 0.05 заряда
 sendData(double): void  Метод, который выводит в консоль информацию о количестве переданных данных (срабатывает, если спутник активен)

Не забудь переопределить методы performMission() и toString() (смотри выше)

4) Класс SatelliteConstellation

Этот класс соответствует спутниковой группировке. Она связана со спутниками с помощью агрегации в методе addSattelite(Satellite satellite)

Что нужно сделать 

  • Так как в Java можно реализовать полиморфизм, нужно поместить в список satellites общие классы родители Satellite, которые внутри себя уже будут хранить информацию о том, какая конкретно реализация сделана

  • В группировке необходимо создать методы работы со спутниками: добавление, получение и выполнение всех миссий

Атрибуты

constellationName: String  Наименование группировки
satellite: <List>Satellite  Список спутников, входящих в группировку

Методы

 addSatellite(Satellite): void  Метод для добавления спутника в группировку
 executeAllMission(): void

 Метод выполнения миссий для всех спутников группировки
 getSatellites(): <List>Satellite  Метод получения списка спутников
